# **App Name**: KrishiAI

## Core Features:

- Multilingual Voice Chat: Enables farmers to interact via voice in multiple Indian languages to ask questions and receive advice. A tool uses the farmer's language choice and location when responding.
- Personalized Crop Advisory: Provides customized crop recommendations based on location, soil type, and historical data. Integrated fertilizer calculator suggests optimal NPK ratios.  The tool uses location and soil to decide what information to present.
- AI Pest & Disease Detection: Allows farmers to upload images of infected plants; the AI analyzes the image and recommends treatments.
- Hyper-Local Weather Alerts: Delivers real-time, location-specific weather forecasts and predictive alerts to the user.
- Market Price Dashboard: Displays current and historical market prices for key crops in nearby markets in a simple table format.
- Offline Access: Allows users to download critical information, such as pest guides and common advisories, for offline use in areas with low connectivity.
- User-Friendly Interface: Features a simple, intuitive interface optimized for low-end smartphones and low bandwidth conditions, prioritizing ease of use for users with low digital literacy.

## Style Guidelines:

- Primary color: Earthy Green (#6B8E23) to symbolize growth and nature.
- Background color: Very Light Beige (#F5F5DC) for a clean and accessible feel.
- Accent color: Golden Yellow (#FFC72C) for important alerts and calls to action, analogous to green but offering strong contrast.
- Body and headline font: 'PT Sans', a humanist sans-serif.
- Use culturally relevant, easily understandable icons representing crops, weather patterns, and advisory topics.
- Prioritize a clean and straightforward layout with large, clear buttons and minimal visual clutter.
- Subtle, non-distracting animations for loading and feedback to minimize data usage.
